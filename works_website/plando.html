Works

有限会社プランドゥ WEBサイト

サイト概要

前職のイベント企画・広告デザイン制作等の会社のWEBサイト。

ターゲット

以下の4つの層をターゲットに制作しております

  • 企業のマーケティング部門
  • イベント主催者
  • 広告デザイン関連企業
  • Web広告を活用したい企業

目的

企業のプロモーション活動を支援し、ブランド価値の向上と顧客満足の最大化を実現すること

課題

お問い合わせフォームを制作し、独学でPHPを導入して実際に動かせることができたが、導入できなかった。

デザイン

TOP画像は企業イメージの画像を使用。E.ドリームのWEBサイトを流用したが、反対の黒をベースにして、無彩色を使うことで統一感を出しました。。

情報設計

上司の指示により、1ページでサイトを作成。色を統一して、一体感を持たせる。業務案内を簡潔に書き、何ができるか一目でわかるように。

制作期間

完成までの時間：約64時間

使用ソフト

Photoshop / Visual Studio Code
