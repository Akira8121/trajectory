Works

E.ドリームプロモーション株式会社 WEBサイト

サイト概要

前職の関連会社、イベント企画・映像制作等の会社のWEBサイト。

ターゲット

以下の4つの層をターゲットに制作しております

  • 企業のマーケティング部門
  • イベント主催者
  • メディア関連企業
  • Web広告を活用したい企業

目的

企業のプロモーション活動を支援し、ブランド価値の向上と顧客満足の最大化を実現すること

課題

コンタクトフォームを設置し、PHP等を使って問い合わせを行えるようにしたいです。

デザイン

シンプルなデザインにし、会社内容を分かりやすく。
一番最初に目に入るTOP部分はインパクトがあるように大きめに配置。目が疲れないように、スクロールは5秒に設定。
4つの主要サービスは、見てもらいやすいように、他より大きな文字・英字にして目立たせる。
footer部分は、ロゴ・社名・メール問い合わせ、事業部情報に分け、今後事業部が増えても追加しやすいように作成。

情報設計

上司からの指示で、1ページでサイトを作成。
一番最初に目に入るTOP部分は事業部のある関東のモノクロ写真をスクロールで配置。 TOP部分のキーワードは白文字にシャドウを付け、モノクロ画像の上で目立つように。
4つの主要サービスを目立たせ、分かりやすいように設計しました。

制作期間

初稿のコード入力：合計約32時間
初稿から完成までの修正時間：合計約150時間

使用ソフト

Photoshop(画像編集) / Visual Studio Code
